# spec.md — ZeroTrust Dynamics | Premium Cybersecurity Landing Page

## 1. Project Overview

**Business Name:** ZeroTrust Dynamics (זירוטראסט דיינמיקס)
**Business Type:** Cybersecurity Startup (סטארטאפ אבטחת סייבר)
**Tone:** Professional, authoritative, cutting-edge, premium dark-mode aesthetic.
**Language:** Hebrew (fully localized — all UI text, headings, descriptions, CTAs, and labels must be in Hebrew).
**Layout Direction:** `dir="rtl"` applied globally on the root `<html>` or top-level `<div>`.

**Design Philosophy:**
This is NOT a generic landing page. This is a cinematic, Awwwards-level digital experience for a high-end cybersecurity brand. Every pixel must communicate trust, sophistication, and technological superiority. Think: dark voids with controlled neon light bleeds, glass surfaces floating over deep gradients, and purposeful micro-animations that reward the user's attention.
