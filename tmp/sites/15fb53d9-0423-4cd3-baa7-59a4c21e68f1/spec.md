# spec.md — מוסך רונן | Premium Hebrew Landing Page Technical Specification

---

## 1. Project Overview

Build a **single-page, Hebrew-first, RTL landing page** for **מוסך רונן** — a professional auto repair garage.
The design must be **Awwwards / Dribbble quality**: cinematic, breathable, and premium.
No basic Bootstrap-era layouts. No flat colors. No generic stock-photo vibes.
Every section must feel intentional, modern, and trust-building.

---

## 2. Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 14 (App Router) |
| Styling | Tailwind CSS v3 |
| Animations | Framer Motion |
| Icons | lucide-react |
| Fonts | Google Fonts — `Heebo` (primary, Hebrew), `Inter` (numeric accents) |
| Language Direction | RTL (`dir="rtl"` on `<html>`) |
