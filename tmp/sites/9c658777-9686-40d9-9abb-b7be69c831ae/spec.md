# spec.md — ZeroTrust Dynamics Premium Landing Page

## 1. Project Overview

**Business Name:** ZeroTrust Dynamics (זירוטראסט דיינמיקס)
**Business Type:** Cybersecurity Startup (סטארטאפ אבטחת סייבר)
**Tone:** Professional, authoritative, cutting-edge, premium
**Language:** Hebrew (fully localized)
**Layout Direction:** RTL (`dir="rtl"`) on the root `<html>` or top-level `<div>`
**Design Philosophy:** Dark-mode-first. Awwwards/Dribbble-tier. Zero tolerance for flat, generic, or "template-looking" design. Every pixel must feel intentional, deep, and premium.

---

## 2. Tech Stack & Dependencies

| Dependency | Purpose |
|---|---|
| **React 18+** (with TypeScript) | Component architecture |
| **Tailwind CSS 3.4+** | Utility-first styling, RTL support via `rtl:` variants |
| **framer-motion** | Scroll-reveal animations, hover micro-interactions, stagger effects, layout animations |
| **lucide-react** | All iconography — no other icon library permitted |
| **Google Fonts: Heebo (400, 500, 700, 800, 900)** | Hebrew-optimized premium sans-serif typeface |
