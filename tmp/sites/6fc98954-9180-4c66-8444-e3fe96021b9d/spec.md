# spec.md — מוסך רוזן פרו | Premium Hebrew Landing Page

---

## 1. Project Overview

A world-class, Awwwards/Dribbble-quality Hebrew landing page for **מוסך רוזן פרו** — a professional auto repair garage emphasizing full transparency, fast service, and fair pricing. The page must feel premium, trustworthy, and modern — far above typical garage websites. Every section must breathe, animate, and delight.

---

## 2. Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 14 (App Router) |
| Styling | Tailwind CSS v3 |
| Animations | Framer Motion |
| Icons | lucide-react |
| Fonts | Google Fonts — `Heebo` (primary, Hebrew-optimized), `Inter` (fallback) |
| Language Direction | RTL (`dir="rtl"`) on `<html>` and all layout wrappers |
