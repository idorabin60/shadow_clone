## 1. Project Overview

Build a **single-page, pixel-perfect, Awwwards/Dribbble-quality** Hebrew landing page for **מוסך רונן** — a professional auto repair garage. The page must feel premium, trustworthy, and modern — far above the typical garage website aesthetic. Think: the design language of a high-end SaaS product, applied to automotive services.

**Stack:**
- React (with Vite or Next.js)
- Tailwind CSS (JIT mode)
- Framer Motion (all animations)
- Lucide React (all icons)
- Google Fonts: `Heebo` (primary) + `Assistant` (body)

**Critical Global Rules:**
- `dir="rtl"` on `<html>` and all major containers — NO exceptions.
- `lang="he"` on `<html>`.
- All text is Hebrew.
- Zero placeholder lorem ipsum — use only the real business copy provided below.
- Mobile-first, fully responsive (375px → 1920px).
- Lighthouse score target: 90+ Performance, 100 Accessibility.
